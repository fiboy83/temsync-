# **App Name**: temsync

## Core Features:

- Holographic Backdrop Generation: Generates unique, blurred holographic backdrops for user posts using AI, ensuring each post has a distinct visual theme.
- Dynamic Post Templates: Offers pre-designed, minimal holographic post templates to streamline content creation, ensuring aesthetic consistency.
- Floating Navigation Bar: Implements a floating bottom navigation bar for easy access to core features: Market, Swap, Post creation, and Inbox.
- User Profile Icons: Displays profile icons in the top right for quick access to user profiles and settings.
- Search Functionality: Integrates a search icon in the top left, enabling users to easily search for content and users.
- Dummy Posts: Populates the feed with seven holographic-themed dummy posts from various users to demonstrate platform usage and aesthetics.

## Style Guidelines:

- Primary color: Electric indigo (#6F00FF) for a modern holographic feel.
- Background color: Light gray (#E0E0E0) with a slight desaturation for a soft backdrop.
- Accent color: Cyan (#00FFFF) to highlight interactive elements.
- Headline font: 'Space Grotesk' sans-serif for a techy feel; body font: 'Inter' sans-serif for readability.
- Use clean, minimalist icons with a subtle holographic glow effect.
- Implement a card-based layout for posts with rounded corners and soft shadows to enhance the floating effect.
- Subtle fade-in animations for loading content and transitions between views.
