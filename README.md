# temsync - holographic social multiverse

**temsync** is a futuristic, AI-powered social media platform built for the holographic age. It features a deep glassmorphism aesthetic, dynamic user-centric themes, and a seamless focus-based content consumption model.

## ✨ features

- **holographic aura**: interface colors dynamically synchronize with the user's profile avatar frequency (hue).
- **glassmorphism UI**: high-contrast, deep backdrop blur effects with sharp, elegant borders.
- **focus mode**: a distraction-free environment for reading posts and engaging with signals (comments).
- **interactive inbox**: real-time simulated transmissions with profile integration.
- **AI-powered**: dummy data and holographic backdrops generated via Genkit flows.
- **lively transitions**: optimized GPU-accelerated animations for a responsive, agile experience.

## 🚀 tech stack

- **framework**: Next.js 15 (App Router)
- **styling**: Tailwind CSS + Shadcn UI
- **icons**: Lucide React
- **AI/GenAI**: Google Genkit (Gemini 2.5 Flash)
- **fonts**: Space Grotesk (Headline) & Inter (Body)

## 🛠️ getting started

1. **install dependencies**:
   ```bash
   npm install
   ```

2. **set up environment**:
   Create a `.env` file with your `GEMINI_API_KEY`.

3. **run development server**:
   ```bash
   npm run dev
   ```

4. **build for production**:
   ```bash
   npm run build
   ```
